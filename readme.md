## Prerequisites

Composer
See install instructions at [getcomposer.org](https://getcomposer.org/doc/00-intro.md)

## Install Sitesettest

Clone from github
``` bash
$ git clone git@github.com:divarsoy/sitesettest.git
```

## Usage
Update vendors with composer

``` bash
$ php composer.phar install
```

## Testing

``` bash
$ phpunit
```
